## 📊 Income Scaling Formula

```typescript
// Income per second at a given level
function getBusinessIncome(def: BusinessDef, level: number): number {
  if (level <= 0) return 0;
  return def.baseIncome * level * 0.1;
}

// Cost to upgrade from level N to level N+1
function getUpgradeCost(def: BusinessDef, currentLevel: number): number {
  return Math.floor(def.baseCost * Math.pow(def.costMultiplier, currentLevel));
}
```

**Examples (Hot Dog Stand — baseCost: $500, costMultiplier: 1.15, baseIncome: $1/s):**

| Level | Upgrade Cost | Income/s | Total Invested |
|-------|-------------|----------|----------------|
| 1 | $500 (buy) | $0.10/s | $500 |
| 5 | $974 | $0.50/s | $3,620 |
| 10 | $2,023 | $1.00/s | $10,597 |
| 20 | $8,183 | $2.00/s | $55,638 |
| 50 | $586,116 | $5.00/s | $2,895,810 |
| 100 | $265,877,718 | $10.00/s | $851M |

**Note:** The income formula scales linearly but costs scale exponentially, creating the classic idle game tension where you need to diversify across businesses.
