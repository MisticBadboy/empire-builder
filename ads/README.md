## 📊 Recommended Ad Strategy

### Phase 1: Organic (Free)
- Post the video on TikTok, Reels, Shorts
- Use relevant hashtags
- Engage with comments

### Phase 2: Paid Boost ($50-100 test)
- **TikTok Ads Manager**: Boost the best-performing organic post
- Target: Ages 18-35, interests in idle games, tycoon games, casual games
- **Meta Ads**: Create "App Install" campaign with this video
- Budget: $10/day for 7 days = $70 test

### Phase 3: Scale (if CPI < $0.50)
- Increase budget on winning platforms
- Create 2-3 more variations of the ad
- A/B test different hooks (first 3 seconds)

## 🔄 Re-rendering the Video

To regenerate with updated content:
```bash
cd ads/
node render-ad.js
```

Edit the HTML files (`p1.html` through `p4.html`) to change text/content.
Each `p*.html` is one scene of the video.
